# Barbarian Prince: Road of Iron

A lightweight browser strategy-adventure inspired by the *Barbarian Prince* board-game premise.

## Play locally

1. Start a static file server:
   ```bash
   python3 -m http.server 8000
   ```
2. Open <http://localhost:8000>.
3. Move through the frontier, survive the campaign clock, rescue the captive royal, defeat the usurper, and return to the capital.

## Highlights

- 8x8 map exploration with fog of war.
- Dynamic terrain events and encounter choices.
- Class system (Warrior, Scout, Chieftain) with gameplay bonuses.
- Multi-stage victory condition (rescue + fortress battle + capital return).
- Save campaign state to browser storage.
